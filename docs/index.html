Getting Started

1
Install Docker Desktop (free) and make sure it's running
2
Download and install the launcher for your platform above
3
Open ProjectBlackVault — it'll set everything up automatically
4
Click Open App in the launcher to start managing your vault
macOS users: If you see a security warning on first launch, right-click the app and choose Open.

Windows users: If SmartScreen warns you, click More info → Run anyway.

All data stays on your computer. Nothing is sent to the cloud.
View source on GitHub →
ProjectBlackVault is open source and self-hosted — your data never leaves your machine.
