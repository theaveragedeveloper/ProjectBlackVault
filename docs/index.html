// Features

🔫
Firearm Inventory
Catalog every firearm with make, model, caliber, serial number, and photos. Serial numbers encrypted at rest.
🛠
Visual Configurator
Call of Duty–style build editor. Assign parts to slots, group by category, track cost per build.
📦
Accessories & Ammo
Manage your optics, suppressors, lights, and ammunition stock. Round counts auto-update from range sessions.
🎯
Range Sessions
Log sessions to track rounds fired, deduct ammo, and keep usage records across all firearms.
🔒
Encryption at Rest
AES-256-GCM encryption for sensitive fields. Key generated automatically by the installer.
🏠
Fully Self-Hosted
Runs locally via Docker. No telemetry, no accounts, no cloud. Your data never leaves your machine.

// Self-Hosting in 3 Steps

Get up and running in minutes

1
Install Docker Desktop
Download and install Docker Desktop for your operating system from docker.com. Ensure it's running before continuing.
2
Run the installer
macOS / Linux: open Terminal and run bash install.sh
Windows: double-click install.bat or run it in Command Prompt.

The installer will prompt for a data directory and port, generate your encryption key, and start the app automatically.
3
Open BlackVault
Navigate to http://localhost:3000 (or the port you chose). Your vault is ready — add your first firearm to get started.
