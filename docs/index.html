Already running this on your network? Remote users connecting over your network or VPN just need the URL — no download required. The launcher is only needed on the machine that runs Docker.

Getting Started

1
Download and run the launcher for your platform above
2
If Docker isn't installed, the launcher will offer to install it automatically (or you can install Docker Desktop manually first)
3
Open ProjectBlackVault — it'll download and set everything up automatically
4
Click Open App in the launcher to start managing your vault
macOS users: If you see a security warning on first launch, right-click the app and choose Open.

Windows users: If SmartScreen warns you, click More info → Run anyway.

All data stays on your computer. Nothing is sent to the cloud.
